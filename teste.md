<h1 align=center> Resumo da apresentação</h1>

## Sharing Ideas: Construindo uma metodologia de aprendizado aberta e colaborativa baseada no GitHub

O atual modelo de ensino é de maneira geral centralizado na figura do professor. Tradicionalmente, o processo de aprendizado flui de maneira passiva do professor para o aluno. Considerando os avanços tecnológicos atuais aliados as discussões sobre a necessidade de se colocar os alunos com protagonistas do próprio aprendizado, o presento projeto propõe uma metodologia que busca apoiar a autonomia do aluno, o trabalho colaborativo baseado em projetos e o compartilhamento conhecimento. Através desse projeto, vamos demonstrar as possibilidades de apoio ao aprendizado usando uma proposta de metodologia de educação distribuída e colaborativa. A plataforma GitHub mais utilizada para hospedagem de código-fonte foi explorada neste projeto visando a adaptação de seus recursos já consolidados no desenvolvimento colaborativo de software para outras áreas do conhecimento. O projeto busca facilitar uma cultura de colaboração e melhoria espontânea, porém estruturada, uma vez que a plataforma em questão reúne cm sucesso equipes para trabalhar com problemas globalmente distribuídos, movendo ideias e aprendizado entre si ao longo do caminho. Para o desenvolvimento da metodologia está sendo observado o comportamento atual e o desejado nos processos de aprendizado sendo este traduzido na forma de interações com os recursos do GitHub. Desta forma, busca-se organizar as estruturas da sala de aula, a fim de facilitar a troca de conhecimento e aprendizado colaborativo.  O foco do trabalho são turmas de forma geral,de diferentes áreas do conhecimento que possuem o potencial de se beneficiar ao romper com amarras da estrutura tradicional de aprendizagem. O diferencial do projeto é adaptar a metodologia de desenvolvimento distribuído de software para áreas que não utilizam código e não estão relacionadas à tecnologia e explorá-lo no contexto do CESJF.
